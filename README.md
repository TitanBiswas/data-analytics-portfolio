# Data Analytics Portfolio

Welcome! This repository brings together five data analytics projects completed with Python. Each project demonstrates a practical workflow: loading data, cleaning it, exploring patterns, visualising results, and documenting useful business insights.

## Projects

| Project | Focus | Tools | Key outcome |
| --- | --- | --- | --- |
| [BigBasket Product Analysis](projects/bigbasket-product-analysis) | Product catalogue exploration, missing values, outliers and visualisation | Pandas, NumPy, Seaborn | Identifies product-level patterns after data-quality checks. |
| [Netflix Content Analysis](projects/netflix-content-analysis) | Streaming catalogue composition, genres, countries and release trends | Pandas, Matplotlib, Seaborn | Shows how Netflix content varies by type, genre, country and time. |
| [Vehicle Insurance Analysis](projects/vehicle-insurance-analysis) | Insurance customer and claim-response analysis | Pandas, Matplotlib, Seaborn | Highlights customer attributes associated with insurance response. |
| [Zomato Restaurant Analysis](projects/zomato-restaurant-analysis) | Indian restaurant landscape, cuisines, cities, ratings and price | Pandas, Matplotlib, Seaborn, Plotly | Finds leading cities, chains, cuisines and pricing patterns. |
| [Spotify Music Analysis](projects/spotify-music-analysis) | Music trends, audio features, genres and artist popularity | Pandas, Matplotlib, Seaborn | Documents trends in energy, danceability, popularity and genre representation. |

## Getting started

1. Clone or download this repository.
2. Create and activate a virtual environment (recommended).
3. Install the dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Open a project notebook with Jupyter:

   ```bash
   jupyter notebook
   ```

5. Start the notebook from its own project folder so its relative dataset path works.

## Data notes

The Netflix and Vehicle Insurance datasets are included. The BigBasket and Zomato source datasets were not included with the notebooks; their project folders explain the exact filenames needed to run them. Review each dataset's licence and source terms before redistributing it publicly.
