# BigBasket Product Analysis

Exploratory data analysis of a BigBasket product catalogue. The notebook imports the dataset, checks its structure and missing values, handles data-quality issues, examines outliers, and visualises product patterns.

## Tools

Python, Pandas, NumPy, Seaborn, and Jupyter Notebook.

## Run the project

1. Place the source dataset in `data/` with this exact filename: `BigBasket Products.csv`.
2. Open `notebooks/bigbasket_product_analysis.ipynb` from this project folder.
3. Run the cells in order.

## Notebook workflow

- Import and inspect product data.
- Identify null values and inconsistent records.
- Treat missing values and outliers where appropriate.
- Explore patterns and create visualisations.

## Data availability

The notebook was supplied, but its CSV file was not. This repository intentionally does not fabricate or substitute the source data.
