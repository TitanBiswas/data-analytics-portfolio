# Vehicle Insurance Analysis

This project analyses vehicle-insurance customer data to identify the factors associated with a positive insurance response. It includes data inspection, duplicate removal, descriptive analysis, and visual exploration.

## Tools

Python, Pandas, NumPy, Matplotlib, Seaborn, and Jupyter Notebook.

## Included files

- `data/Vehicle_Insurance.csv` - customer and policy dataset.
- `notebooks/vehicle_insurance_analysis.ipynb` - complete analysis notebook.

## Key insights

- Older vehicles show a higher probability of a claim response.
- Customers with prior vehicle damage are more likely to respond.
- Previously insured customers respond less frequently.
- Annual premium has a weak relationship with response.
- Response frequencies differ across regions.

## Run the project

Open the notebook while this project folder is the working directory. If necessary, update the dataset path to `data/Vehicle_Insurance.csv`.
