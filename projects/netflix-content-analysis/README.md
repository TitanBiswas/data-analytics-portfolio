# Netflix Content Analysis

This project explores the Netflix titles catalogue to understand its content mix, leading genres and countries, release-year trends, ratings, and movie durations.

## Tools

Python, Pandas, NumPy, Matplotlib, Seaborn, and Jupyter Notebook.

## Included files

- `data/netflix_titles.csv` - Netflix catalogue dataset.
- `notebooks/netflix_content_analysis.ipynb` - complete analysis notebook.

## Key findings

- Netflix has a diverse library, with strong representation from selected genres and countries.
- New-title additions vary substantially across years.
- Most movie durations fall roughly between 80 and 120 minutes.
- Genre popularity changes over time.

## Run the project

Open the notebook while this project folder is the working directory. If needed, change the CSV path in the notebook to `data/netflix_titles.csv`.
