# Zomato Restaurant Analysis

An exploratory analysis of the Indian restaurant landscape using Zomato data. The project cleans a large restaurant dataset and studies geographical distribution, chains, establishment types, cuisines, prices, ratings, delivery, and takeaway availability.

## Tools

Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly, and Jupyter Notebook.

## Key findings

- Chennai, Mumbai, Bengaluru, and Pune have the largest restaurant counts in the analysis.
- Domino's Pizza and Cafe Coffee Day are among the largest chains by outlet count.
- Quick Bites and Casual Dining are the most common establishment types.
- North Indian cuisine has the largest representation, followed by Fast Food.
- Most restaurants fall in the mid-range cost-for-two segment, and ratings are commonly between 3 and 4.

## Run the project

1. Add the source file to `data/` as `Indian-Resturants.csv` (the notebook's original spelling).
2. Open `notebooks/zomato_restaurant_analysis.ipynb` from this project folder.
3. Run the notebook cells in order.
